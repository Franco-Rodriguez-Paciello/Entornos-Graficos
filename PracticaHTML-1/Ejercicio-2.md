# Entornos-Graficos Practica-1 Ejercicio-2

2.a) 
Este segmento nos muestra como se representan Los comentarios en HTML. Estos pueden colocarse en cualquier parte del documento, tanto en la sección <head> como en el <body>, o incluso fuera del contenido visible.
Este es solo visible durante la etapa del desarrollo y no tiene implicancia dentro de la pagina, por lo tanto no es obligatorio. No tiene etiquetas ni atributos.

2.b)
Este segmento nos muestra una etiqueta <div>. Esta representa un contenedor genérico de bloque. Este código se coloca en la sección <body> del documento HTML. Porque representa contenido visible en la página web, y todo lo que se muestra al usuario debe ir dentro del <body>.
El atributo en este ejemplo es id, no es obligatorio. Permite identificar este div de forma única dentro del documento.

2.c)
Este segmento Este segmento nos muestra una etiqueta <img> se coloca dentro de la sección <body>, ya que forma parte del contenido visual de la página. Se usa para insertar una imagen en el sitio web.

-Atributos: src, alt, id, name, width, height, longdesc.
-Valores: src="", alt="lugar imagen", id="im1", name="im1", width="32", height="32", longdesc="detalles.htm"

El unico atributo obligatorio es src, como esta vacio no se mostrare imagen si no el atributo alt.

2.d)

Este segmento Este segmento nos muestra una etiqueta <meta> se coloca dentro de la sección <head>. Son metadatos los cuales ya fueron mencionados en el ejercicio 1

-Atributos: name, lang, content, http-equiv
-Valores: name="keywords", lang="es", content="casa, compra, venta, alquiler ", http-equiv="expires", content="16-Sep-2019 7:49 PM"

Los atributos name y content son obligatorios.

2.e)

 Este segmento Este segmento nos muestra una etiqueta <a> se coloca dentro de la sección <body>. Establece un hipervinculo que tiene el texto "Resumen HTML"
-Atributos: href, type, hreflang, charset, rel
-Valores: href="http://www.e-style.com.ar/resumen.html" type="text/html" hreflang="es" charset="utf-8" rel="help"

El atributo href es obligatorio.

2.f)

Este segmento nos muestra una etiqueta <table> se coloca dentro de la sección <body>. Se encarga de marcar la creacion de una tabla.
Este segmento nos muestra una etiqueta <caption> se coloca dentro una <table>. Se encarga de definir un título o leyenda para la tabla..
Este segmento nos muestra una etiqueta <th> se coloca dentro de <table>, <thead>, <tbody> o <tfoot>. Se encarga de definir una fila de una tabla.
Este segmento nos muestra una etiqueta <tr> se coloca dentro de una fila <tr>. Se encarga de definir una celda de encabezado..

-Atributos: width, summary, align, scope
-Valores width="200" summary="Datos correspondientes al ejercicio vencido" align="top" cope="col" scope="row"

No hay atributos obligatorios
